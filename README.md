## 学習内容（概要）

### 基本環境
- Linux学習用環境の準備（EC2 / VM）
- ユーザー・権限周りの基本確認
- 学習ログ用ディレクトリ構成の整理

### シェル・スクリプト
- 基本的なシェル操作（変数 / リダイレクト / パイプ）
- if / for / while の動作確認
- 簡単なシェルスクリプト作成
- 実行権限（chmod）の確認と記録

### ネットワーク基礎
- IPアドレス・サブネットの確認
- ip / ss / ping コマンドの実行ログ
- ルーティングテーブルの確認とメモ

### システム管理
- systemdサービスの状態確認
- サービスの起動・停止・自動起動設定
- ログファイルの確認（journalctl）

### セキュリティ・権限
- ファイルパーミッションの確認
- sudo設定の確認
- 簡単なセキュリティ設定の整理

### ストレージ
- ディスク・パーティション構成の確認
- マウント設定の確認
- df / lsblk の結果を記録
